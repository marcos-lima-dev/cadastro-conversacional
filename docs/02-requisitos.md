# 🧩 O que precisamos para fazer?

Para construir esse projeto, vamos utilizar tecnologias modernas focadas em performance, experiência do usuário e escalabilidade.

---

## 🧠 Stack principal

### Frontend
- Next.js (App Router)
- React
- TypeScript

---

## 🎨 UI e Experiência

- Tailwind CSS (estilização)
- Lucide Icons (ícones)
- Framer Motion (animações)

---

## 🧪 Validação e Formulário

- React Hook Form (controle de formulário)
- Zod (validação de dados)

---

## ⚙️ Funcionalidades necessárias

### 1. Fluxo conversacional
- Perguntas em sequência
- Controle de etapa atual

### 2. Validação de dados
- CPF
- Telefone
- E-mail
- Data

### 3. Feedback de erro
- Mensagens simples e humanas
- Destaque visual

### 4. Exibição dos dados preenchidos
- Cards laterais
- Atualização em tempo real

### 5. Edição de respostas
- Permitir voltar e alterar dados

### 6. Progresso
- Barra de progresso ou indicador

# 🎯 O que queremos fazer?

Queremos criar um **formulário conversacional moderno** para cadastro de alunos.

Em vez de um formulário tradicional (cheio de campos de uma vez), o usuário será guiado por uma experiência em formato de conversa, onde:

- Cada pergunta aparece individualmente
- O usuário responde uma informação por vez
- Após confirmar, a resposta vira um **card visual na tela**
- O processo continua até o cadastro completo

---

## 🧠 Objetivo principal

Transformar um processo burocrático em uma experiência:

- Simples
- Fluida
- Intuitiva
- Amigável para qualquer tipo de usuário

---

## 💬 Como funciona a experiência

1. O sistema saúda o usuário
2. Faz perguntas guiadas (uma por vez)
3. Valida a resposta
4. Mostra feedback claro em caso de erro
5. Ao confirmar, a resposta aparece na tela como um "resumo"
6. O fluxo continua até o final

---

## ✨ Diferenciais

- UX conversacional (não parece formulário)
- Validação com linguagem humana
- Feedback imediato
- Visualização dos dados em tempo real
- Possibilidade de editar respostas facilmente

---

## 🎯 Resultado esperado

Um protótipo funcional que pode evoluir para:

- Sistema real de matrícula
- Integração com backend
- Uso em escolas, eventos ou cadastros diversos
